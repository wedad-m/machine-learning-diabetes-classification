# Machine Learning Diabetes Classification

This project aims to predict whether a person has diabetes based on medical features such as glucose level, blood pressure, BMI, and age.

## Machine Learning Problem
- **Type:** Classification
- **Target Variable:** Outcome
- **Target Values:**
  - 1 → Diabetic
  - 0 → Non-Diabetic

## Dataset
- **Name:** Pima Indians Diabetes Dataset
- **Source:** Kaggle
- **Format:** CSV
- **Type:** Tabular data

## Project Contents
- Jupyter Notebook for loading and inspecting the dataset
- Methodology diagram illustrating the machine learning workflow
